# AI-SDK-OPENAI

[![AI-SDK Ecosystem](https://img.shields.io/badge/AI--SDK-ECOSYSTEM-part%20of-blue)](https://github.com/mk-knight23/AI-SDK-ECOSYSTEM)

> **Framework**: OpenAI SDK

Production-ready OpenAI SDK implementation with modern web stack.

## Tech Stack

- **Frontend**: Angular 19
- **Backend**: Go Fiber
- **AI SDK**: OpenAI

## Quick Start

```bash
# Frontend
cd frontend
npm install
ng serve

# Backend
cd backend
go run main.go
```

## Features

- OpenAI Assistants API
- RESTful API endpoints
- Docker containerization
- CI/CD pipeline
- Test coverage configured
